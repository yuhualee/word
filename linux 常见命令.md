* **man:** 显示指定命令的用法和描述，如:

	```
	man ls
	``` 


* **touch,cat和less** 

	* touch:创建大小为0的任意类型文件


		```
		touch demo.txt
		```

	* cat: 查看文件内容，不支持分页

		```
		cat filename  //查看文件内容
		cat f1 f2 > f3  //合并文件
		```
	
	* less: 浏览文件，支持上下键查看文件的开头和末尾，more命令类似，但只支持enter翻页，不回退

		```
		less filename
		more filename
		```
